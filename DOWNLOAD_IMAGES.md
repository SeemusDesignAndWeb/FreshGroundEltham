# Download Images from Unsplash

To download coffee-themed background images from Unsplash, run:

```bash
node download-images.mjs
```

This will download the following images to `static/images/`:

- `coffee-beans-background.jpg` - Background for menu section
- `coffee-shop-interior.jpg` - Background for events section
- `coffee-latte-art.jpg` - Gallery image
- `coffee-making.jpg` - Gallery image
- `coffee-cup-hero.jpg` - Hero section background
- `pastries.jpg` - Gallery image
- `coffee-bar.jpg` - Gallery image

These images are already referenced in the components and will automatically appear once downloaded.

## Alternative: Manual Download

If the script doesn't work, you can manually download images from Unsplash:

1. Visit https://unsplash.com/s/photos/coffee
2. Search for coffee-related images
3. Download and save them to `static/images/` with the filenames listed above
